Gerenciador de Itens - Projeto FastAPI + Jinja2

Como executar:
1. Instale dependências:
   pip install fastapi uvicorn jinja2 python-multipart

2. Rode o servidor:
   uvicorn main:app --reload

3. Abra no navegador:
   http://127.0.0.1:8000

Descrição:
- Projeto mantém funcionalidade CRUD em memória (lista 'itens').
- Templates Jinja2: base.html, index.html, create.html, edit.html
- Estilo moderno em static/style.css
